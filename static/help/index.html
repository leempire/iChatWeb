主页
聊天
1. 聊天记录可以保留多久？
为了保护用户隐私，仅提供一天内的聊天记录漫游功能。
2. 怎么发送图片？
发送指令"/img 图片链接"即可发送图片。若没有图片链接，可先将图片上传至云盘，点击"分享"即可获取分享链接。
3. 怎么发送超链接？
发送指令"/iframe 链接 [提示语句]"，即可发送超链接。若不填提示语句，则默认为链接网址。
4. 怎么设置头像？
注册时绑定QQ邮箱的用户默认头像为QQ头像，暂不支持修改头像。
云盘
1. 云盘限制容量吗？
由于服务器内存有限，仅支持上传16Mb以内的文件。
2. 云盘的权限规则是什么？
用户可跨设备查看自己的文件，分享链接发送后所有人可以通过链接下载文件，分享链接在用户删除文件后失效。
3. 怎么分享文件？
部分移动设备页面宽度不足，手动滑动操作栏即可看到"分享"按钮。
