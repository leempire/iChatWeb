主页 游戏 公告 帮助 排行榜 攻略
粒子
1. 粒子可以合成生物、升级混沌（提升雷电生产效率）、合成物质等；
2. 原子核可以生产粒子，高级生物可以生产低级生物；
3. 升级复制后，合成生物数量翻倍；
4. 升级生产后，生物生产效率翻倍。
雷电
1. 雷电可以合成生物、物质等；
2. 使用粒子、尘埃升级混沌、物质、奥秘来提高雷电生产效率；
3. 蓄电池可以储存雷电。
尘埃
1. 物质可以生产尘埃，尘埃用于提升闪电生产效率；
2. 升级复制后，合成物质数量翻倍；
3. 升级生产后，物质生产效率大幅提高；
智慧
1. 粒子达到3.33e12解锁智慧；
2. 思维和神经可以提高智慧生产效率；
3. 升级大脑可以提高智慧的容量；
4. 智慧可以用来研究科技以获得大量资源，也可以用来开启宇宙大爆炸；
5. 科学可以提高科技效果。
暗物质
1. 混沌达到40级或物质达到80级后开启暗物质；
2. 升级混沌和物质可以获得暗物质；
3. 使用智慧引爆宇宙大爆炸（在其他地方消耗将减少爆炸需要）；
4. 宇宙大爆炸后，将在新的世界重新开始，并将暗物质转化为暗能量；
5. 暗能量会在新的世界内触发暗能力，使世界变得更加强大（每10个暗能量+1雷电/s）。
