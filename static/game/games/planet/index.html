操作说明：

1. 单击屏幕可以超指定方向吐球，从而借助反作用力获得反向速度，但是要注意吐球后自己会变小；

2. 两球碰撞时，大球会吸收小球，并获得吸收部分的动量。

3. 吃掉场中所有的星球即可获得胜利；

4. 使用鼠标滚轮缩放画面；

5. 按P键暂停/开始，按左键/右键可减速/加速；

6. 按F键全屏/取消全屏，按Q键退出。

星球类型：

1. 蓝色——玩家星球；

2. 黑色——普通星球；

3. 绿色——反物质星球，与其他球碰撞时，碰撞部分消失；

4. 红色——反引力球，对其他球产生斥力。
