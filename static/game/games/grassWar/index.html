操作说明：

1. 移动鼠标，角色会朝着鼠标的方向前进，被敌人触碰会损失生命；

2. 角色会朝着鼠标方向自动发射子弹，当距离达到子弹的追踪范围时，子弹将朝着敌人的方向发射；

3. 按空格键可以额外发射一级子弹；

4. 随着时间推移，敌人的属性会增加，种类也会增多；

5. 按数字1-4可以强化角色的攻击力、移动速度、生命值、子弹等级，强化所需的金币数量在屏幕左上角相应括号内；

6. 按P键暂停/开始，按F键全屏/取消全屏，按Q键退出。

攻击特效说明：

1. 减速：每击中一次，减少敌人20%移动速度，减速效果可叠加，最低减少至16；

2. 追踪：实时调整方向，瞄准最近的目标；

3. 爆炸：击中敌人后爆炸，对300范围内的敌人造成50%的爆炸伤害；

4. 穿透：可穿透敌人，每穿透一个敌人，伤害提升20%.
